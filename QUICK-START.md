# ⚡ Quick Start - Share & Schedule

## 🚀 התחלה מהירה

---

## 1️⃣ **פתח את הקובץ**
```
flygen-FULL-v4-SHARE-SCHEDULE.html
```
פשוט לחץ לחיצה כפולה או פתח בדפדפן.

---

## 2️⃣ **בדוק שהכל עובד**

### ✅ Test Share:
1. לחץ **עריכה** בדשבורד
2. לחץ **שתף** בתחתית
3. לחץ **WhatsApp** → אמור לפתוח WhatsApp
4. לחץ **Copy** → אמור להעתיק קישור

### ✅ Test Export:
1. לחץ **פרסם קמפיין**
2. לחץ **המשך** (בלי לבחור פלטפורמה)
3. בחר **PDF להדפסה**
4. לחץ **ייצא וסיים**
5. PDF אמור להתחיל להוריד! 📥

### ✅ Test Preview:
1. לחץ **תצוגה מקדימה**
2. אמור להיפתח מסך מלא עם הפלייר
3. לחץ **סגור** או מחוץ למסך

---

## 3️⃣ **כפתורים חדשים ב-Bottom Bar:**

```
┌─────────────────────────────────────────────────────┐
│ [שמור טיוטה]  |  [שתף] [הורד] [תצוגה] [פרסם]      │
└─────────────────────────────────────────────────────┘
```

- **שמור טיוטה** → שומר ב-LocalStorage
- **שתף** → פותח Share Modal
- **הורד** → הורדה מהירה כ-PDF
- **תצוגה מקדימה** → Fullscreen Preview
- **פרסם קמפיין** → Schedule & Export

---

## 4️⃣ **Google Drive (אופציונלי)**

**כרגע:** מציג הודעה ומוריד כ-PNG

**להפעלה מלאה:**
1. צור Google Cloud Project
2. הפעל Drive API
3. עדכן CLIENT_ID בשורה 2537
