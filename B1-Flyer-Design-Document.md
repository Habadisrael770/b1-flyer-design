## 1.6 רכיבי UI (Components)

### 1.6.1 כפתורים (Buttons)

**Primary Button**
- רקע: Purple Primary (#7C3AED)
- טקסט: White
- Hover: Purple Dark (#5B21B6)
- Border Radius: 8px
- Height: 48px
- Padding: 12px 24px

**Secondary Button**
- רקע: White
- טקסט: Purple Primary
- Border: 2px solid Purple Primary
- Hover: רקע Purple Light עם opacity 10%

**Ghost Button**
- רקע: Transparent
- טקסט: Purple Primary
- Hover: רקע Purple Light עם opacity 10%

**Danger Button**
- רקע: Error Red (#EF4444)
- טקסט: White
- Hover: Red Dark (#DC2626)

**מצבים:**
- Default
- Hover
- Active/Pressed
- Disabled (opacity 50%)
- Loading (עם spinner)

---

### 1.6.2 שדות קלט (Input Fields)

**Text Input**
- רקע: White
- Border: 1px solid Medium Gray (#E2E8F0)
- Border Radius: 8px
- Height: 48px
- Padding: 12px 16px
- Focus: Border Purple Primary, shadow outline

**מצבים:**
- Default
- Focus
- Error (border red, הודעת שגיאה מתחת)
- Disabled
- With Icon (אייקון בצד ימין)

**Search Input**
- כמו Text Input
- אייקון זכוכית מגדלת בצד ימין
- כפתור X לניקוי

---

### 1.6.3 כרטיסים (Cards)

## 1.8 אנימציות ומעברים

- Duration: 200ms (מהיר), 300ms (רגיל), 500ms (איטי)

## 2.2 Flow 1: הרשמה והתחברות

### 2.2.1 הרשמה חדשה

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Splash    │────▶│   Login     │────▶│  Register   │────▶│  Dashboard  │
│   Screen    │     │   Screen    │     │   Form      │     │             │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘

                          │ Google OAuth
                          ▼
                    ┌─────────────┐
                    │   Google    │
                    │   Popup     │
                    └─────────────┘
```

**שלבים:**
1. **Splash Screen** (2 שניות) - לוגו + אנימציית טעינה
2. **מסך התחברות** - אפשרויות:
   - התחבר עם Google (כפתור ראשי)
   - או הרשמה עם Email
3. **טופס הרשמה:**
   - שם מלא
   - אימייל
   - סיסמה
   - אישור סיסמה
   - שם העסק (אופציונלי)
4. **אימות** - שליחת קוד לאימייל (אופציונלי)
5. **מעבר לדשבורד**

## 2.4 Flow 3: יצירת פליירי

### 2.4.1 יצירת פליירי חדש

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  Flyers     │────▶│   Select    │────▶│   Select    │────▶│   Flyer     │
│   List      │     │  Template   │     │  Products   │     │   Editor    │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
                                                                   │
                    ┌─────────────┐     ┌─────────────┐            │
                    │   Share/    │◀────│   Preview   │◀───────────┘
                    │   Export    │     │   Flyer     │
                    └─────────────┘     └─────────────┘
```

**שלב 1: בחירת תבנית**
- גלריית תבניות מוכנות
- תבניות לפי קטגוריה (מבצעים, חגים, כללי)
- תצוגה מקדימה של כל תבנית
- אפשרות "תבנית ריקה"

**שלב 2: בחירת מוצרים**
- רשימת כל המוצרים
- חיפוש וסינון
- בחירה מרובה (Checkbox)
- הגבלה: 3-12 מוצרים לפליירי
- מונה מוצרים נבחרים

**שלב 3: עורך הפליירי**
- Canvas עם התבנית
- גרירת מוצרים למקומות
- עריכת כותרת
- שינוי צבעים
- שינוי גופנים
- הוספת לוגו
- הוספת פרטי קשר

**שלב 4: תצוגה מקדימה**
- תצוגה מלאה של הפליירי
- אפשרות לחזור ולערוך
- אישור ושמירה

**שלב 5: שיתוף/ייצוא**
- הורדה כ-PNG/PDF
- שיתוף ישיר לוואטסאפ
- שיתוף לרשתות חברתיות
- שליחה במייל

### 4.1 מסך התחברות (Login Screen)

#### מטרה
לאפשר למשתמשים קיימים להתחבר ולמשתמשים חדשים להירשם

#### אלמנטים

| אלמנט | סוג | תיאור | התנהגות |
|-------|-----|-------|---------|
| לוגו | תמונה | לוגו B1-Flyer | מרכז למעלה |
| כותרת | טקסט | "מערכת יצירת פליירים חכמה" | תחת הלוגו |
| כפתור Google | Button Primary | "המשך עם Google" + אייקון | OAuth login |
| מפריד | קו + טקסט | "או" | --- |
| טאבים | Tab Group | "התחברות" / "הרשמה" | מחליף טופס |
| שדה אימייל | Input Field | placeholder: "הכנס אימייל" | validation |
| שדה סיסמה | Input Field | placeholder: "הכנס סיסמה" | type: password |
| שדה שם (הרשמה) | Input Field | placeholder: "שם מלא" | רק בהרשמה |
| כפתור שליחה | Button Primary | "התחבר" / "הירשם" | submit form |
| שכחתי סיסמה | Link | "שכחת סיסמה?" | רק בהתחברות |

#### Validation Rules
- אימייל: פורמט תקין (xxx@xxx.xxx)
- סיסמה: מינימום 6 תווים
- שם: מינימום 2 תווים

#### Error States
- "אימייל לא תקין"
- "סיסמה קצרה מדי"

- "סיסמה שגויה"

## 9. מערכת שיתוף (Share System)

### 9.1 Share Modal

#### מטרה
לאפשר למשתמש לשתף את הפלייר בפלטפורמות שונות

#### אלמנטים

| אלמנט | סוג | תיאור | פעולה |
|-------|-----|-------|-------|
| קישור שיתוף | Input (readonly) | URL ייחודי לפלייר | מוצג אוטומטית |
| WhatsApp | Button | ירוק + אייקון | פותח WhatsApp עם הודעה מוכנה |
| Facebook | Button | כחול + אייקון | פותח שיתוף לפייסבוק |
| Email | Button | אדום + אייקון | פותח מייל עם קישור |
| Copy Link | Button | סגול + אייקון | מעתיק קישור ללוח |
| Download | Button | אפור + אייקון | הורדה מהירה כ-PNG |

#### URL Format
```
https://flygen.app/share/flyer-{timestamp}
```

#### התנהגות כפתורים

**WhatsApp:**
```
פותח: https://wa.me/?text=צפה בפלייר שלי: {URL}
```

**Facebook:**
```
פותח: https://www.facebook.com/sharer/sharer.php?u={URL}
```

**Email:**
```
פותח: mailto:?subject=פלייר מבצעים&body=צפה בפלייר: {URL}
```
